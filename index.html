SkillSet

Java 개발자 양성 과정을 수료하면서 웹 개발에 필요한 기초를 학습했습니다. 과정을 마치고 나서는 Java를 기반으로 다양한 개념과 기술을 익히고 정리하고 있습니다. 또한 Spring Framework를 활용하여 구현한 프로젝트를 업데이트 중입니다. 이 프로젝트를 통해 Spring Framework에 대한 경험을 쌓으면서 지속적으로 스프링의 다양한 기능과 개념을 학습하고 있습니다

Java

자바의 기초를 공부하고 자료구조와 데이터 구조를 학습했습니다. 자바를 활용하여 객체지향적인 코드를 개발하는 능력을 향상시키고 있으면 이를 통해, 미니프로젝트를 구현했습니다

Spring

Spring으로 MVC디자인 패턴을 활용하여 웹서비스를 개발하였습니다. 스프링 데이터 JPA를 활용하고 스프링 Security Oauth를 사용해서 구글 및 카카오 로그인을 구현했습니다.

Oracle SQL

Oracle의 기본 쿼리문을 활용하여 데이터베이스를 사용할 수 있습니다. 대부분의 프로젝트에서 Oracle를 사용해봤고 배포중인 프로젝트에는 OracleDB를 사용하고 있습니다.

HTML5

HTML 페이지를 설계하고 Thymeleaf 템플릿 문법을 이용해서 데이터를 원하는 위치에 동적으로 넣을 수 있습니다.

CSS3

CSS를 활용하여 개인 홈페이지 프로젝트를 진행했었습니다. 또한, tailwind css를 활용하여 다른 프로젝트도 원활히 진행하였습니다.

IntelliJ

IntelliJ IDE의 다양한 기능을 활용하여 코드작성, 디버깅, 리팩토링을 효과적으로 프로젝트에 적용시켰습니다.

GitHub

GitHub를 활용해서 프로젝트를 관리하고 커밋을 세분화하는 훈련을 했습니다. GitHub를 사용해서 학습한 내용도 저장하고 있습니다.

Notion

Notion을 활용하여 Agile 방법론을 적극적으로 도입하고 실행했습니다. 이 도구를 사용하여 프로젝트 관리, 작업 분류, 스프린트 계획 및 진행 상황 추적 등의 Agile 핵심 원칙을 효율적으로 구현했습니다.

현대 사회에서 1인 가구가 증가하는 추세에 따라 이들에게 필요한 정보와 서비스를 제공하기 위해 "One_Zip"이라는 프로젝트를 진행했습니다. 스프링 부트를 기반으로 CRUD를 구현하고, 로깅프레임워크, JPA, 스프링 시큐리티 등 새롭게 학습한 내용을 기반으로 작업하였습니다. 주요기능으로는 커뮤니티 및 쇼핑몰 기능이 있습니다.
